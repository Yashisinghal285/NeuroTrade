# 🧠 NeuroTrade

NeuroTrade is a smart stock news alert system that fetches real-time news related to stocks or companies and sends instant updates through a Telegram bot. It helps users stay informed and avoid major financial losses by reacting quickly to important news.

## 🚀 Features
- Real-time stock news fetching
- Telegram bot notifications
- Fast alerts for market-moving events
- Simple and lightweight system

## 🛠️ Tech Stack
- Python
- News API
- Telegram Bot API

## 📌 How It Works
1. User selects a stock
2. System fetches latest news
3. Filters important updates
4. Sends alerts via Telegram
